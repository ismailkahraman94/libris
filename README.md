# Library Management System

A beautiful desktop application for managing your personal library.

## Features

- **Multi-User Profiles:** Create multiple profiles to manage separate libraries on the same device.
- **Dashboard:** View all your books with cover images, reading status, and progress.
- **Add Book:** Automatically fetch book metadata (Title, Author, ISBN, Cover, Summary) from Kitapyurdu and Google Books API.
- **Shelf Management:** Organize your books into custom shelves.
- **Reading Tracking:** Track your reading progress, start/finish dates, and current page.
- **Beautiful UI:** Modern Material Design interface using Flet.

## Installation

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Run the application:
```bash
python main.py
```

## Technologies

- Python
- Flet (UI)
- SQLite (Database)
- BeautifulSoup4 (Scraping)
